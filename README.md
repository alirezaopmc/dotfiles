# dotfiles
## Setup

Auto installation:
```bash
chmod +x setup.sh
./setup.sh
```

Manutal installation:
```bash
git clone https://github.com/alirezaopmc/dotfiles.git
cd dotfiles
ln -s "neovim" "~/.config/nvim"
ln -s "wezterm" "~/.config/wezterm"
ln -s "tmux/.tmux.conf" "~/.tmux.conf"
ln -s "zsh/.zshrc" "~/.zshrc"
ln -s "bash/.bashrc" "~/.bashrc"
```
